# Bales Guitars

Quality electric guitars assembled and professionally set up in Scottsboro, Alabama.

## Overview

Bales Guitars specializes in taking high-quality guitar kits and elevating them through precision setup, fretwork, and professional finishing. Each guitar is carefully assembled, set up, and tested for optimal playability before being made available.

## What We Do

### Our Process
- **Precision Fretwork**: Leveling, crowning, and polishing frets for a smooth, broken-in feel
- **Professional Setup**: Truss rod adjustment, action setting, and intonation dialing
- **Quality Control**: Clean wiring, cavity shielding, and hardware upgrades where needed
- **Play Testing**: Every guitar is tested for playability before listing

### Guitar Styles
- **ST Style**: S-S-S configuration with modern finishes
- **TL Style**: S-S configuration with classic aesthetics
